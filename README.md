This is a Stardew Valley mod that allows teleportation by clicking on the map, based on Aedenthorn's mod, compatible with version 1.6.

### Known Bugs:
SVE，RSV，ES模组传送坐标错误
Incorrect teleport points for SVE, RSV, ES
